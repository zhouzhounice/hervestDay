## 复杂度分析

常见的时间复杂度（递增）

- 常量阶：O(1)
- 对数阶：O(logn)
- 线性阶：O(n)
- 线性对数阶：O(nlogn)
- 平方阶：O(n^2)
- 立方阶：O(n^3)
- k次方阶：O(n^k)
- 指数阶：O(2^n)  --非多项式量级
- 阶乘阶：O(!n)  --非多项式量级

把非多项式量级的问题称为__NP(non-deterministic polynomial)__问题

### O(1)

```c
int i = 8;
int j = 6;
int sum = i + j;
```

### O(logn)、O(nlogn)

```c
i = 1
    while(i <= n){
        i = i * 2;
    }
```

### O(m+n)、O(m*n)

```c

int cal(int m, int n) {
  int sum_1 = 0;
  int i = 1;
  for (; i < m; ++i) {
    sum_1 = sum_1 + i;
  }

  int sum_2 = 0;
  int j = 1;
  for (; j < n; ++j) {
    sum_2 = sum_2 + j;
  }

  return sum_1 + sum_2;
}
```
